## AWS LAMBDA GRAPHQL SERVERLESS

### Installation

1. Clone the repository

2. Install the packages

```
npm install
```
### Serverless offline  
To run serverless offline comment the `- serverless-webpack` in the serverless.yml file and run 

```
serverless offline start
```
